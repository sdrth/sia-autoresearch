## Observations

- Fact: every "verticalize early" success story examined had a functioning activation/onboarding motion first; the failures verticalized to escape a weak core
- Fact: vertical template packs appear in both the verticalization literature and the onboarding literature — same artifact, two payoffs
- Inference: for this product, vertical templates are a two-for-one — they test vertical demand and attack the run-001 blank-state problem
- Inference: the real decision is not vertical vs horizontal but which falsifiable vertical test to run first
- Gap: no current data on which industries existing users actually come from — bets below start there
